# 🎨 Color Flipper

A simple JavaScript project that changes the background color of the page with a click.

---

## 🚀 Features

* Change background color on button click
* Random color generation
* Instant UI update

---

## 🛠 Tech Stack

* HTML
* CSS
* JavaScript

---

## 📂 Project Structure

```
color-flipper/
│── index.html
│── style.css
│── script.js
```

---

## ▶️ How to Run

1. Download or clone this repo
2. Open `index.html` in your browser
3. Click the button
