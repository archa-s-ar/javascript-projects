# 🎨 Color Flipper

An interactive JavaScript project that changes the background color of the webpage using buttons.

This project demonstrates core JavaScript concepts like DOM manipulation, event handling, and dynamic styling.

---

## 🚀 Features

* 🎯 Change background color using predefined buttons (Green, Blue, Red, Random)
* 🎲 Generate random colors dynamically
* ⚡ Instant UI updates
* 🎨 Clean and simple user interface

---

## 🛠 Tech Stack

* HTML
* CSS
* JavaScript

---

## 📂 Project Structure

```
color-flipper/
│── index.html
│── style.css
│── script.js
```

---

## ⚙️ How It Works

* Buttons trigger JavaScript functions using click events
* `setColor(color)` changes the background to a specific color
* `randomColor()` generates random RGB values and applies them

Example:

```
rgb(123, 45, 200)
```

---

## 💡 Key Concepts Learned

* DOM selection (`getElementsByTagName`)
* Event handling (`onclick`)
* Functions in JavaScript
* Random number generation (`Math.random`)
* Template literals (`` `rgb(${})` ``)
* Dynamic styling with JavaScript

---

## ▶️ How to Run

1. Clone or download this repository
2. Open `index.html` in any browser
3. Click the buttons to change colors 🎉
